# dev.twin.life website

[![License](http://img.shields.io/badge/license-APACHE2-blue.svg)](LICENSE)

This is the dev.twin.life website implemented on top of [Ada Web Application](https://github.com/stcarrez/ada-awa).

# Build

Building this application requires to have [Alire](https://github.com/alire-project/alire) installed.

```
git clone --recursive https://github.com/Twinlife/dev-website.git
cd dev-website
alr update
make
```
